Saving Your Work

To save your work, go to the "Workspace" menu and press "Save". A window will open with a text area containing Hax workspace data. In this version there is no storage backing the web page so you have to save the data yourself.

To open an existing workspace, press "Open" from the "Workspace" menu. Paste in the data from the "save" command and press "Open".

There is also a version of Hax for desktop running in Electron. This allows you to save to and open from your local file system.
