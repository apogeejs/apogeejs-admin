• Open the Hax web application at http://haxapp.com/hax.html
  • Create a new Data Table called "settings"
    • Right click in the main window.
    • Select "Add Data Table" from the context menu.
    • Enter the name "settings" and press the "create" button.
  • Set a value for this table.
    • Option 1: Enter the JSON value as text:
      • Select "JSON" from the dropdown on the table "settings".
      • Click in the table body to enter edit mode.
      • Paste in the following JSON data: 
        {
	"initialYear": 2015,
	"initialRevenue": 1000000,
	"growthRate": 3,
	"projectionCount": 10
}
      • Press save to save the data.
    • Option 2: Enter the data using the Form mode:
      • Select "Form" from the dropdown on the table "settings".
      • Right click in the table body to get a context menu. Select "Convert to Object"
      • Type in the key and value data from the JSON above.
  • Create a new Data Table called "revenueList".
  • Add a formula for the new table.
    • From the dropdown on the table select "Formula".
    • Click the window body to enter edit mode.
    • Paste in the following code. You can drag the table edge to make the table bigger. 
      value = [];
value[0] = settings.initialRevenue;
for(var i = 1; i < settings.projectionCount; i++) {
    value[i] = value[i-1] * (1 + settings.growthRate/100);
}
    • Press "save".
    • From the dropdown, select "Text". A JSON array should be displayed.
