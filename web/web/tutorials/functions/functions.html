Tutorial - Functions

There are two ways to add functions in Hax: Simple Functions and Folder Functions.

Simple Functions

Plain javascript functions can be added as tables to the workspace.

  1. Start with a new project. Open a new window with the url http://haxapp.com/hax.html
  2. Close the default workspace and create a new workspace called "FunctionTest"
  3. To create a function, use "Add Function" from the context menu.
  4. Name the function "foo" and set the argument list to "x,y".
  5. Create a data table called "b". Do this by right clicking inside the window for folder "folder1". The dialog box to create table "b" should show the folder as "FolderTest:folder1".

    Notice the color of the border of the folder1 inner window. This should match the color of the outer border of table "b", and of all objects that it contains.
