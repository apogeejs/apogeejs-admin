Interactive Examples

These examples illustrate basic usage of the application.

  • Basic Example - Basic table values and formulas.
  • "Immutable" Values Gotcha - This example shows a when you need to make an explicit copy of data.
  • Folders and Scope - This illustrates variable scope with regards to folders.
  • The Grid and Spreadsheets - This example shows a grid table and how to use it to import and export spreadsheet data.
  • External Libraries and Controls - How to link external libraries and controls.
  • Asynchronous Formulas (The Promise Land) - Using asynchronous formulas, like web requests.
  • Don't Kill the Messenger - How to push data to other tables, a seeming cheat on "no side effects"
  • Custom Controls - Making a custom control right in your workspace.
  • Folder Functions - Writing a function with tables, rather than lines of code.
  • Combined Example - A bigger example, converting and rendering Open Street Map data.
