Apogee is a spreadsheet-inspired programming tool designed for data processing, programmatic JSON editing and general software coding.

Spreadsheet Apogee
Best for: data entry, data presentation, simple calculations. Best for: data processing, programmatic JSON editing.
  • A grid of cells, each cell is named according to its location in the grid.
  • A cell value is a simple value like a number or a string.
  • Each cell has either static value or a formula to calculate this value.
  • Apogee replaces cells with individual tables. Tables are given descriptive names such as "revenue".
  • A table value is a data structure such as a number, an array or something more complex.
  • Each table has a static value or a formula to calculate the value.

Motivation for Apogee

The motivation behind Apogee was to make a reactive functional programming environment like a spreadsheet but to allow for more complex code and easier maintenance.

  • The variables names from a spreadsheet, such as B$5, are replaced with descriptive names such as revenue to make the code more understandable.
  • The formulas are applied to entire data structures, so loops or recursive functions can be used, rather than copying and pasting formulas over a range of cells.

Implementation

The Apogee web application runs completely in the browser and is written in javascript. The user scripting language is also javascript.

Because Apogee is uses javascript, the data format for data structures is JSON. This is a convenient because JSON is one of the most prevelent data formats, such as for metadata files and data transfer in web services.

Compatibility

  • Desktop Chrome, Firefox and Edge browsers tested. Safari also supported.
  • Internet Explorer not supported
  • Desktop support only. Touch not supported.
