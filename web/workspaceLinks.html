Sample Workspace Links

This page contains links to sample Hax workspaces. These links are to the raw JSON file. Temporary instructions to open one:

  • Copy the JSON contents from the link.
  • From the Hax application, click on the menu entry to Workspace > Open
  • Paste the workspace JSON into the dialog box and press open.
