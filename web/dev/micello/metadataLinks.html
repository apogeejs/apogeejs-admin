Metadata Links

Meta Data Edit Web Applications

  • Set Community Com File - This specifies the com override file for a community.
  • Edit Com File - This edits community override files.
  • Edit Meta File - This edits meta data base or override files.
  • Create File Entry - This creates a file entry, which compose the entries in a meta file.
  • Upload Com File - This upload a community override file to MFS.
  • Upload Meta File - This uploads a metadata override or base file to MFS.

Meta Data Utilities

  • Create Customer Json - This creates and updates a customer JSON. The customer JSON must be manually maintained and uploaded to the link: https://eng.micello.com/apogee/data/customerJson.json
  • Update Metadata DB - This can be used for a general update of the Micmetar DB.
  • Open Community Overrides - View the current community override files for a community.

General Utilities

  • Set Community Scale - This updates the community scale for CAD input. It should be done before the CAD file is imported.
  • Fix single nodes links in a CID - This deletes any single node links in a given community.
  • Update Micemaps DB - This can be used for a general update of the Micemaps DB.
  • Update Micentity DB - This can be used for a general update of the Micentity DB.
