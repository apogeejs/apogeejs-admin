Meta Data Edit Workspaces

These projects are used to create the meta data files and deploy them to MFS.

To use these:

  1. Navigate to the folder tasks.login and login. From there you will be guided through the edit process.
  2. On leaving the page, ignore any prompt for saving the workspace. There is nothing here to save.
  • Edit Meta or Com File - This edits community override files and meta data base and override files.
  • Create File Entry - This creates a file entry, which compose the entries in a meta file.
  • Upload Meta File - This uploads a metadata override or base file to MFS.
  • Upload Com File - This upload a community override file to MFS.

Utilities

  • Create Customer Json - This creates and updates a customer JSON. The customer JSON must be manually maintained and uploaded to the link: http://eng.micello.com/apogee/data/customerJson.json

Map Links

  • Set Community Scale - This updates the community scale for CAD input. It should be done before the CAD file is imported.
